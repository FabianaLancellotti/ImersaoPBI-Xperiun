# 📊 Imersão Power BI para Negócios

Este repositório contém os projetos desenvolvidos durante a **Imersão Power BI para Negócios**. Ao longo do curso, foram criadas soluções de Business Intelligence para três áreas críticas de uma empresa: Finanças, Logística e Vendas.

O foco principal foi o domínio do ciclo completo de um projeto de BI: Extração, Tratamento (ETL), Modelagem de Dados, Cálculos com DAX e Design de Dashboards (UX/UI).

## 📁 Conteúdo do Repositório

### 1. [Dashboard Financeiro](./Dashboard-Financeiro)
* **Foco:** Gestão de fluxo de caixa e controle de despesas.
* **Destaques:** Tratamento de dados no Power Query, criação de menu de navegação e análise de categorias (Fixo vs. Variável).

### 2. [Dashboard de Logística](./Dashboard-Logistica)
* **Foco:** Performance operacional e monitoramento de entregas.
* **Destaques:** Cálculo de *Lead Time*, análise de ocorrências e devoluções, e visão por tipo de frota e filiais.

### 3. [Dashboard de Vendas](./Dashboard-Vendas)
* **Foco:** Inteligência comercial e performance de equipe.
* **Destaques:** Modelagem de dados avançada (Star Schema), análise de lucratividade e uso de *Tooltips* (Dicas de Ferramenta) visuais.

---

## 🛠️ Tecnologias Utilizadas
* **Microsoft Power BI:** Desktop e Service.
* **Power Query:** Para limpeza e transformação de dados.
* **Linguagem DAX:** Para criação de métricas de negócio.
* **Excel/CSV:** Bases de dados utilizadas para os cenários reais.

## 🚀 Como Visualizar os Projetos
Para explorar os dashboards, você pode baixar os arquivos `.pbix` localizados dentro de cada pasta e abri-los no Power BI Desktop.
